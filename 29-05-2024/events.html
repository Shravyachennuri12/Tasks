What are Events?
Events are actions or occurrences that happen in the browser that the browser can respond to. Examples include clicking a button, loading a page, or pressing a key.
Event Types Here are some common types of events in JavaScript:
Mouse Events: These occur when the user interacts with the mouse.
click: When the user clicks on an element.
dblclick: When the user double-clicks on an element.
mouseover: When the user moves the mouse over an element.
mouseout: When the user moves the mouse out of an element.
Keyboard Events: These occur when the user interacts with the keyboard.
keydown: When the user presses a key.
keyup: When the user releases a key.
keypress: When the user presses a key (deprecated).
